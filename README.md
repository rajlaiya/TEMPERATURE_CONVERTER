Temperature Converter

    This is a Temperature Converter website designed to convert temperatures between Celsius and Fahrenheit. The site includes a real-time conversion feature and changes the background based on the temperature value inputted by the user. The design is sleek, responsive, and includes smooth animations for a better user experience.

Features

    Real-Time Temperature Conversion: The website instantly converts temperatures between Celsius and Fahrenheit as the user enters the values.

    Dynamic Background: The background changes dynamically based on the temperature entered, creating an immersive experience.

    Responsive Design: The website adapts to various screen sizes, providing a consistent experience on desktop, tablet, and mobile devices.

    Smooth Animations: The website includes animations that make transitions between pages and temperature changes visually appealing.

Technologies Used

    HTML: The structure and content of the website.

    CSS: Styling and layout, including the glass effect and responsive design.

    JavaScript: Handles real-time temperature conversion and dynamic background changes.

Animation: 

    CSS animations for smooth transitions and interactive user experience.
